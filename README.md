## Windows

### Building - Visual Studio IDE 2019

```
premake5 vs2019
```

Double click on the generated `Morpheus.sln` file.

## Linux

```
$ premake5 gmake2
```

### Building

```
$ make
```

### Running

```
./bin/x64_Debug/Premake_Example
```
